# 주제

+ 프로젝트 명 : 자격증 시험 정보 제공 챗봇
+ 자격증 시험에 대한 정보를 쉽게 얻을 수 있게 하기 위한 챗봇
  + 자격증 시험 일정 및 시험 접수
  + 응시료 및 준비물

# 기대효과
+ 자격증에 대한 정보를 손쉽게 파악
+ 자격증 시험 일정을 파악하여 일정 관리에 용이
+ 사용자 입력 정보를 통한 많이 찾는 자격증 확인
